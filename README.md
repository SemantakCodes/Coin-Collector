# 🟨 Cool Cube Collector

Cool Cube Collector is a simple 3D endless runner game built in Unity where the player controls a cube, collects coins, and avoids obstacles to achieve the highest score possible.

---

## 🎮 Gameplay

- Control a cube that moves forward continuously
- Move left and right to collect coins
- Avoid obstacles to stay alive
- The game gets progressively harder over time
- Try to beat your high score!

---

## 🚀 Features

- Endless runner mechanics
- Smooth player movement
- Coin collection system
- Obstacle spawning system
- Score tracking
- Increasing difficulty over time
- Simple and clean UI

---

## 🛠️ Built With

- **Unity** (3D)
- **C#**
- **Unity Input System**

---

## 🎯 Controls

| Action     | Key / Input        |
|------------|--------------------|
| Move Left  | `A` / `← Left Arrow` |
| Move Right | `D` / `→ Right Arrow` |
| Jump       | `Space`            |
